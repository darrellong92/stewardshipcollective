Connect your Google Sheet to make this calendar live

Currently showing built-in data. To connect Google Sheets so updates appear automatically:

1. Google Sheets → File → Share → Publish to web → "Events" sheet → CSV → Publish → copy the URL.
2. Open this HTML file, find the line below, and paste your URL between the quotes:

const GOOGLE_SHEET_CSV_URL = 'PASTE-YOUR-PUBLISHED-CSV-URL-HERE';

3. Save and re-upload to Netlify. All future updates happen in Google Sheets only — no code changes needed.
